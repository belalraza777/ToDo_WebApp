# 📝 To-Do Web App with Authentication

A full-stack To-Do List web application built with **Node.js**, **Express.js**, **MongoDB**, and **EJS**, featuring a secure authentication system using **JWT** and **bcrypt**. Users can sign up, log in, and manage their personal tasks securely.

---

## 🚀 Features

### 🔐 Authentication
- User **Signup/Login** with hashed passwords (`bcrypt`)
- **JWT**-based authentication for session management
- **Protected routes** for authenticated users only

### ✅ Task Management
- **Add**, **edit**, **delete**, and **mark** tasks as complete/incomplete
- Each user has their **own task list**
- User-friendly UI with **EJS templates**

---

## 🛠 Tech Stack

- **Frontend**: EJS (Embedded JavaScript Templates)
- **Backend**: Node.js, Express.js
- **Database**: MongoDB with Mongoose
- **Authentication**: JWT, bcrypt

### 📦 Key Packages
- `express`
- `mongoose`
- `jsonwebtoken`
- `bcrypt`
